# starter-ts-mongo-express
Starter Kit Mongo DB TypeScript and express js

## Description

This is a starter kit for a Mongo DB TypeScript and express js application.

## Requirements

### Node.js

    npm install

### Tech Stack

    TypeScript
    Mongoose
    Express
    Morgan
    Body-parser
    Passport


## Installation

    npm install

## Usage

    npm start

    http://localhost:3000

    npm test

    npm run dev
